Namespace: Config

Config

Config example to pass into phaserControls
Properties:
Name Type Description
name string The name of the control scheme
controls object Controls names and keys to use
active string Whether to set this control scheme to active
Since:
  • 1.0.0
Source:
  • phaserControls.js, line 62

Example

const config = {
    name: 'azertyKeys',

    controls: {
        up: 'Z',
        down: 'S',
        left: 'Q',
        right: 'D',
        shift: 'SHIFT',
        space: 'SPACE',
    },

    active: false,
}
